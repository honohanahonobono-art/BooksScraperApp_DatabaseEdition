# 書籍仕入れ判断を支援する業務改善アプリ

このアプリは、書籍の仕入れ判断を効率化するための
業務支援ツールです。

カテゴリ別に評価情報を可視化し、
「どのジャンルを優先して仕入れるべきか」を
判断しやすくします。

想定利用者：
・小売店スタッフ
・営業事務
・仕入れ担当者

利用シーン：
・仕入れ会議前の事前確認
・経験に頼らない判断材料の作成

使用技術：
・Python
・pandas
・Streamlit
・SQLite

工夫した点：
・ITに不慣れな人でも使えるUIを意識
・業務でよくある「確認作業の手間」を削減
・データは保存・再利用できる設計にした
